## Tentang Aplikasi

ShotSync adalah aplikasi Manajemen Produksi Film yang membantu tim produksi mengelola project film, scene, task, dan koordinasi tim secara efisien.

## Fitur Utama

- **Manajemen Proyek** - Kelola multiple project film dengan detail lengkap
- **Scene Management** - Atur scene-by-scene dengan informasi lokasi, equipment, task, dan jadwal
- **Task Tracking** - Assign dan monitor task untuk setiap anggota tim
- **Personal Notes** - Catat catatan penting secara lokal
- **AI Chat Assistant** - Bantuan AI untuk planning dan problem-solving produksi
- **Real-time Chat** - Komunikasi real-time antar anggota tim
- **Notifications** - Notifikasi untuk update task dan reminder task
- **Map Integration** - Visualisasi lokasi shooting dengan map viewer
- **Biometric Security** - Keamanan login dengan autentikasi biometrik
- **Profile Management** - Kelola profil dan avatar pengguna

## Teknologi

- **Framework:** Flutter (Dart)
- **Backend/Database:** Supabase (PostgreSQL)
- **Authentication:** Biometric Authentication, Supabase Auth
- **Maps:** Flutter Map Integration, OpenStreetMap, Nominatim API
- **Local Storage:** Shared Preferences, Hive
- **Notifications:** Flutter Local Notifications
- **State Management:** Getx State Management
- **Media:** Image Picker, Camera
- **Platform:** Android, iOS
